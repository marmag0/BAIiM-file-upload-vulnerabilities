# BAIiM-file-upload-vulnerabilities
A fast-paced overview of File Upload Vulnerabilities, covering both the theoretical background (impact, attack vectors, bypass techniques) and practical, step-by-step exercises to help understand, reproduce, and mitigate insecure file upload scenarios.


### Requirements

- PortSwigger account
- Burp Suite
- Internet access


### Lab Links

If you don't want to clone this repo to access links in pdfs, you can access labs directly though links below:
- [Unrestricted File Upload](https://portswigger.net/web-security/file-upload/lab-file-upload-remote-code-execution-via-web-shell-upload)
- [Flawed Identification of File Type](https://portswigger.net/web-security/file-upload/lab-file-upload-web-shell-upload-via-content-type-restriction-bypass)
- [Obfuscation of File Extension](https://portswigger.net/web-security/file-upload/lab-file-upload-web-shell-upload-via-obfuscated-file-extension)
- [Exploiting File Upload Race Conditions](https://portswigger.net/web-security/file-upload/lab-file-upload-web-shell-upload-via-race-condition)
